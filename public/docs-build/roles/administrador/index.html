Guía para Administrador

Tu Rol en el Sistema

Section titled “Tu Rol en el Sistema”

Como Super Admin, tienes control total del sistema:

  • Gestión de usuarios y roles
  • Configuración de dependencias y áreas
  • Acceso completo a todas las afiliaciones
  • Administración del sistema
  • Auditoría y monitoreo

Acceso Completo

Section titled “Acceso Completo”

Todas las Acciones

Section titled “Todas las Acciones”
MóduloAcciones
AfiliacionesCrear, ver, editar, eliminar, validar, rechazar, restaurar
UsuariosCrear, ver, editar, eliminar, asignar roles
DependenciasCrear, ver, editar, eliminar
ÁreasCrear, ver, editar, eliminar
RolesVer y asignar permisos (Shield)
AuditoríaVer todos los logs de actividad

Gestión de Usuarios

Section titled “Gestión de Usuarios”

Crear Usuario

Section titled “Crear Usuario”
  1. Ve a Usuarios en el menú
  2. Click en Crear
  3. Completa la información:
  1. Click en Crear

Asignar Roles

Section titled “Asignar Roles”

Los roles disponibles son:

  • super_admin: Acceso total
  • SSST: Valida afiliaciones
  • Dependencia: Crea afiliaciones

Cambiar Rol de Usuario

Section titled “Cambiar Rol de Usuario”
  1. Ve a Usuarios
  2. Edita el usuario
  3. Cambia el campo Roles
  4. Guardar

Desactivar Usuario

Section titled “Desactivar Usuario”

Para usuarios que ya no necesitan acceso:

  1. Edita el usuario
  2. Marca como inactivo (si hay campo) o elimina
  3. El usuario no podrá iniciar sesión

Section titled “Gestión de Dependencias”

Crear Dependencia

Section titled “Crear Dependencia”
  1. Ve a Dependencias
  2. Click en Crear
  3. Completa:
  1. Click en Crear

Desactivar Dependencia

Section titled “Desactivar Dependencia”

Si una dependencia se elimina o reorganiza:

  1. Edita la dependencia
  2. Desmarca Activo
  3. Guardar

Note

Las afiliaciones existentes se mantienen, pero no se pueden crear nuevas para esta dependencia.

Gestión de Permisos (Shield)

Section titled “Gestión de Permisos (Shield)”

Acceder a Shield

Section titled “Acceder a Shield”
  1. Ve a Shield en el menú (si está visible)
  2. O usa el comando: php artisan shield:generate --all

Estructura de Permisos

Section titled “Estructura de Permisos”

Cada recurso tiene permisos:

  • view_any_afiliacion: Ver lista
  • view_afiliacion: Ver detalle
  • create_afiliacion: Crear
  • update_afiliacion: Editar
  • delete_afiliacion: Eliminar
  • force_delete_afiliacion: Eliminar permanente
  • restore_afiliacion: Restaurar

Crear Rol Personalizado

Section titled “Crear Rol Personalizado”

Si necesitas un rol intermedio:

  1. Accede a la gestión de roles
  2. Crea nuevo rol
  3. Asigna permisos específicos
  4. Guarda

Ejemplo: Rol “Supervisor” que solo puede ver pero no editar.

Resolución de Problemas

Section titled “Resolución de Problemas”

Usuario No Puede Acceder

Section titled “Usuario No Puede Acceder”
  1. Verifica que el usuario existe
  2. Verifica que tiene rol asignado
  3. Verifica que el rol tiene permisos
  4. Limpia caché de permisos

Afiliación Atascada

Section titled “Afiliación Atascada”

Si una afiliación tiene problemas de estado:

  1. Edita directamente en la base de datos
  2. O usa tinker para modificar

Error en Importación

Section titled “Error en Importación”

  2. Verifica formato del Excel
  3. Verifica que las dependencias existan
  4. Limpia y reintenta

Checklist de Seguridad

Section titled “Checklist de Seguridad”
